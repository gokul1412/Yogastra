# Yogastra - Yoga & Meditation Center HTML Template

Yogastra is a premium, responsive, static HTML website template designed for Yoga & Meditation Centers. It features a modern, calming aesthetic with dark/light mode and RTL support, built using semantic HTML5, CSS3, Bootstrap 5, and vanilla JavaScript.

## Features
- 11 Public Pages & 6 Member Dashboard Pages
- Fully Responsive (Mobile-first)
- Dark/Light Mode Toggle
- RTL Support
- WCAG 2.1 AA Accessibility
- Form Validation
- SEO Friendly

## Technology Stack
- HTML5
- CSS3 (Custom Properties / Variables)
- Bootstrap 5
- JavaScript (ES6+)
- Google Fonts (Lora, Nunito)
- Font Awesome Icons
